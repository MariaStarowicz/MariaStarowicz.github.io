spooky run


a small project created for the purpose of completing the 'python programming' course at my university. this simple game was created in collaboration with a friend. the font and sounds for the game were taken from free assets. the graphics were created by us using adobe photoshop and illustrator.

'spooky run' is a game with mechanics similar to the 'chrome dinosaur game'. the player avoids spiders and bats in order to get as many points as possible. after hitting an obstacle, the game is over.
