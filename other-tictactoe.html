tic tac toe


this project is a cross platform app which is a tic-tac-toe game. the project was created in xamarin, it runs on android and universal windows platform. the game includes 3 levels of difficulty.

in easy mode, the computer marks a random place on the board
in medium mode, the computer checks rows, columns and diagonals and marks a spot if it finds 2 opponent's crosses or circles, and if it doesn't find any opponent's pairs of selections, marks a random spot.
in hard mode you can at most tie with the computer. it is not possible to win because the minimax algorithm is used there, which always finds the optimal move for the computer.
