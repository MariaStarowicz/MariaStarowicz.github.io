space invaders remake


this 2D game was created by me shortly after I started learning programming in unity. this project is a remake of the space invaders game. in order to win, player has to shoot down all the opponents. the game contains a simple script for saving the ranking to a text file.

music for this project were downloaded from epidemicsound.com. graphics were made in adobe illustrator and sound effects were created in bfxr.
